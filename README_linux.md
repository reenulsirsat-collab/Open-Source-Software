## How to Run

### Prerequisites
- Linux system (Ubuntu/Debian/CentOS)
- Bash shell
- Python3 installed (for script2)

### Execution Steps
```bash
chmod +x *.sh
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
```
